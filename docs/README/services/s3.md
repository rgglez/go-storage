# s3 — Amazon S3

[![Go Reference](https://pkg.go.dev/badge/github.com/rgglez/go-storage/services/s3/v3.svg)](https://pkg.go.dev/github.com/rgglez/go-storage/services/s3/v3)
[![Status](https://img.shields.io/badge/status-stable-brightgreen)](https://github.com/rgglez/go-storage)

Backend for [Amazon S3](https://aws.amazon.com/s3/) and S3-compatible services (MinIO, DigitalOcean Spaces, Wasabi, etc.).

## Install

```bash
go get github.com/rgglez/go-storage/services/s3/v3
```

## Usage

```go
import (
    "github.com/rgglez/go-storage/services/s3/v3"
    ps "github.com/rgglez/go-storage/v5/pairs"
)

// Amazon S3
srv, sto, err := s3.New(
    ps.WithCredential("hmac:<access_key>:<secret_key>"),
    ps.WithName("<bucket_name>"),
    ps.WithLocation("us-east-1"),
    ps.WithWorkDir("/optional/prefix/"),
)

// S3-compatible (e.g. MinIO, DigitalOcean Spaces)
srv, sto, err := s3.New(
    ps.WithCredential("hmac:<access_key>:<secret_key>"),
    ps.WithEndpoint("https://nyc3.digitaloceanspaces.com"),
    ps.WithName("<bucket_name>"),
    ps.WithLocation("nyc3"),
    s3.WithForcePathStyle(true),   // needed by some S3-compatible services
)

// Storager only
sto, err := s3.NewStorager(
    ps.WithCredential("hmac:<access_key>:<secret_key>"),
    ps.WithName("<bucket_name>"),
    ps.WithLocation("us-east-1"),
)
```

## Configuration

| Pair | Type | Required | Description |
|------|------|----------|-------------|
| `credential` | string | Yes | Authentication. Format: `hmac:<access_key>:<secret_key>` |
| `name` | string | Yes | Bucket name |
| `location` | string | Yes | AWS region (e.g. `us-east-1`, `eu-west-1`) |
| `endpoint` | string | No | Custom endpoint for S3-compatible services |
| `work_dir` | string | No | Working directory (key prefix). Defaults to `/` |
| `force_path_style` | bool | No | Use path-style URLs instead of virtual-hosted. Useful for MinIO/localhost |
| `use_accelerate` | bool | No | Enable S3 Transfer Acceleration |
| `use_arn_region` | bool | No | Use the region in the ARN of the bucket access point |

## Storage Classes

| Constant | S3 Class |
|----------|----------|
| `s3.StorageClassStandard` | STANDARD |
| `s3.StorageClassReducedRedundancy` | REDUCED_REDUNDANCY |
| `s3.StorageClassGlacier` | GLACIER |
| `s3.StorageClassStandardIa` | STANDARD_IA |
| `s3.StorageClassOnezoneIa` | ONEZONE_IA |
| `s3.StorageClassIntelligentTiering` | INTELLIGENT_TIERING |
| `s3.StorageClassDeepArchive` | DEEP_ARCHIVE |

```go
n, err := sto.Write("path/to/key", r, size,
    s3.WithStorageClass(s3.StorageClassStandardIa),
)
```

## Server-Side Encryption

### SSE-S3 (managed keys)

```go
n, err := sto.Write("path", r, size,
    s3.WithServerSideEncryption(s3.ServerSideEncryptionAes256),
)
```

### SSE-KMS

```go
n, err := sto.Write("path", r, size,
    s3.WithServerSideEncryption(s3.ServerSideEncryptionAwsKms),
    s3.WithServerSideEncryptionAwsKmsKeyID("arn:aws:kms:..."),
)
```

### SSE-C (customer-provided key)

```go
n, err := sto.Write("path", r, size,
    s3.WithServerSideEncryptionCustomerAlgorithm(s3.ServerSideEncryptionAes256),
    s3.WithServerSideEncryptionCustomerKey([]byte{...32 bytes...}),
)
```

## Default Pairs (applying SSE globally)

```go
sto, _ := s3.NewStorager(
    ps.WithCredential("hmac:<ak>:<sk>"),
    ps.WithName("<bucket>"),
    ps.WithLocation("us-east-1"),
    s3.WithDefaultStoragePairs(s3.DefaultStoragePairs{
        Write: []types.Pair{
            s3.WithServerSideEncryption(s3.ServerSideEncryptionAes256),
        },
        Read: []types.Pair{},
    }),
)
```
